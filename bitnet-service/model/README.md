# BitNet Model Directory

This directory contains the BitNet model file for local inference.

## Download Model

```bash
cd bitnet-service/model
wget https://huggingface.co/microsoft/bitnet-b1.58-2B-4T-gguf/resolve/main/ggml-model-i2_s.gguf
```

The model file (`ggml-model-i2_s.gguf`) is excluded from git due to size, but this directory structure is maintained.
